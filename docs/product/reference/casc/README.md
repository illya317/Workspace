# 企业会计准则参考文件

来源：CASC 官网 https://www.casc.org.cn/2025/1231/278935.shtml

## 文件清单

1. **企业会计准则第30号-财务报表列报-修订征求意见稿.pdf**
   - 资产负债表、利润表、现金流量表、所有者权益变动表的列报要求
   - 2025年12月征求意见稿，基于IFRS18，引入利润表"五分类"

2. **一般企业利润表的列报格式和列报方法.pdf**
   - 准则正文 + 利润表标准格式

3. **金融企业利润表的列报格式和列报方法.pdf**
   - 金融企业特殊格式

4. **合并利润表的列报格式和列报方法.pdf**
   - 合并报表格式

5. **起草说明.pdf**
   - 修订背景、过程、主要内容
