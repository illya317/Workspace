## Part 一：中间体

测试项: **性状 → 水分 → 含量**

| # | 测试项 | 方法 | 子步骤 |
|---|--------|------|--------|
| 1 | 性状 | 目测 | 实验环境 → 操作方法 → 标准规定 → 异常处理 → 结论 |
| 2 | 水分 | 水分 | +仪器设备 → 操作方法 → 标准规定 → 异常处理 → 清场 → 结论 |
| 3 | 含量 | HPLC/UV | +试验材料+标准品+原始数据 |

> 特例：盐酸左氧氟沙星胶囊、阿奇霉素胶囊无性状（从水分开始）
> 特例：盐酸特拉唑嗪胶囊无含量

---

## Part 三：成品

| # | 测试项 | 方法 | 子步骤 |
|---|--------|------|--------|

## 微生物限度检查

独立结构，字段定义见 `template_fields.yaml`。

```
检验前确认 → 文件 → 实验环境 → 仪器设备 → 进入人员 → 仪器设备(洁净区)
→ 检验用具 → 试验材料 → 标准品 → 检测
  → 大肠埃希菌 → 菌液制备/供试品/阳性/阴性/培养/成立条件/观察
  → 微生物限度 → 供试品/阴性/培养/观察/需氧菌/霉菌
→ 标准规定 → 异常处理 → 清场 → 退出 → 结论
```
